About MFM Launcher

Description

Just click the icon on your desktop, and select a MFM page to open it in your browser.

System

You just need a Windows computer with CMD.EXE on it. CMD.EXE is the terminal application for Windows and is found in the System32 folder. If your computer has MS-DOS Prompt, then it is too old, as MS-DOS prompt will run the Batch file as a DOS Batch, not a Windows Batch. But you shouldn't have to worry about that unless your computer is running something like Windows 95 or something.
